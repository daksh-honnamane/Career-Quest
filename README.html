Career Quest

Career Quest is an interactive career exploration game that turns career research into playable worlds. Instead of only reading about jobs, players step into a workday, complete profession-specific challenges, and leave with a clearer understanding of what each career involves and which skills matter most.

Project Goal

Give players a hands-on "job tour" through multiple careers by combining short workplace scenarios, mini-games, and skill tracking into one cohesive experience.

By the end of a session, players do not just see career names. They experience the pressure, decisions, priorities, and abilities that shape each role.

Quick Facts
  • Built in Python with Pygame
  • Packaged Windows executable included
  • Three complete playable career worlds
  • Local save system with skill tracking

Gameplay Flow

1

Startup Introduction

The game opens with a guided introduction explaining the purpose of Career Quest and preparing the player for career exploration.

2

Career Hub

Players enter the central hub and choose one of three available career worlds: Software Developer, Chef, or Doctor.

3

Career Overview

Each world begins with an overview of the job and highlights the real skills needed to succeed in that profession.

4

Scenario And Challenges

Players move through dialogue scenes and hands-on mini-games designed around realistic responsibilities from that career.

5

Results And Reflection

At the end of a world, the game summarizes performance and records the player's strongest demonstrated skills.

How To Run

Recommended For Judges
  1. Open the project folder.
  2. Launch Career Quest.exe.
  3. Play through the startup scene, enter the hub, and select any career world.

This is the simplest way to evaluate the project on Windows.

Run From Source
  1. Install Python.
  2. Install Pygame with:
pip install pygame
  1. From the project root folder, run:
python -m source.main
